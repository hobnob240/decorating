# Law and Son Decorators Website

This is a static multi-page website built for GitHub Pages.

## Files
- `index.html` — homepage
- `about.html` — about page
- `services.html` — services page
- `gallery.html` — gallery page
- `areas.html` — service areas page
- `enquiry.html` — enquiry page
- `assets/` — CSS, JS, images
- `robots.txt` and `sitemap.xml` — SEO support files
- `.nojekyll` — prevents GitHub Pages from running unwanted Jekyll processing

## Before publishing
1. Replace the placeholder form action in `enquiry.html`
2. Add real project photos
3. Add real testimonials
4. Update the canonical URLs if your final domain changes
5. If you use a custom domain, create a `CNAME` file with your domain
